# 🧠 Sudoku Solver using CSP

## 📌 Features

- Solves 9x9 Sudoku puzzles
- Uses **Backtracking Search**
- Applies **Constraint Checking**
- Tracks:
  - Backtrack Calls
  - Failures

---

## ⚙️ Technologies Used

- Python 3

---

## 📂 Project Structure

AI-Sudoku-CSP/
│── 22F-3242.py        # Main solver code  
│── numbers.txt        # Input Sudoku board  
│── README.md          # Project documentation  

---

## 📥 Input Format

- File contains 9 lines  
- Each line has 9 digits (0–9)  
- `0` represents empty cells  

### Example:

004030050  
609400000  
005100489  
000060930  
300807002  
026040000  
453009600  
000004705  
090050200
